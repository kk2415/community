## :information_desk_person: Overview
### 주요 기능
+ :circus_tent: **스터디/프로젝트 모집**  
  + 간편하게 스터디와 프로젝트 모임을 개설하고 원하는 모임을 신청할 수 있어요

  + 확인하기 귀찮은 채용 공고, 편하게 알림으로 확인하세요!
